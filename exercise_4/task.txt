                                    Практическое занятие по PHP к лекции 4.

1. Напишите функцию, которая будет показывать теперешнее время, время 5 минут назад и время 5 минут вперед.
   Примерно в таком формате:
    -	5 минут назад было: 15:10;
    -	Сейчас: 15:15;
    -	Через 5 минут будет: 15:20.

2. Напишите функцию, которая обработает этот массив:

       [1 => 2, 2 => 3, 3 => 4, 5 => 4, 4 => 6, 6 => 7, 7 => ‘test’]

	Обработать его необходимо следующим образом:
●	Выведите количество элементов в этом массиве в произвольной форме;
●	Удалите последний элемент из массива;
●	Отсортируйте массив по убыванию;
●	Умножьте все значения в массиве, ключи которых нечетные на 2, а четные на 3.
●	Выведите результат на экран.

3. Напишите функцию, которая подключиться к серверу MySQL:
	IP: 192.168.10.206
	Login: root
	Password: testpass123

	А также подключиться к базе данных с названием ‘test-DB’.

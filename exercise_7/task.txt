                                Практическое задание к лекции 7

1.	Напишите SQL запросы, которые будут добавлять в таблицы (INSERT), написанные для предыдущего задания,
    произвольные 10 строк, включая попытки записать туда NULL.

2.	Напишите запросы, которые будут передавать в вашу таблицу невалидные данные
    (строки больше указанной длины, числа неподходящего формата и пр.). Сделайте выводы по
    ограничениям типов данных в базах данных.

3.	Напишите SQL запросы, обновляющие данные следующим образом:
    -   Увеличьте всем персонажам уровень на 10 единиц.
    -	Увеличьте всем персонажам уровень на 5 единиц, но только в том случае, если уровни персонажа не
        более 25 уровня.
    -	Поменяйте классы двум любым персонажам (id 1 => mage, id 2 => warrior на id 1 => warrior, id 2 => mage)
    -	Обновите ники всех персонажей дописав к ним через дефис слово Slayer, если их уровень выше 100

4.	    Напишите SQL запросы, удаляющие:
    -	всех персонажей, которые принадлежат пользователям, возраст которых менее 12 лет.
    -	классы персонажей, которые не принадлежат ни одному персонажу

                                    Практическое занятие по PHP к лекции 6.


1.	Напишите SQL запрос на создание следующих таблиц (типы данных необходимо выбрать
    самим, исходя из логических и экономических соображений):

    Таблица users:
        -	id
        -	name
        -	sname
        -	age
        -	login
        -	pass_hash

    Таблица players:
        -	id
        -	nickname
        -	lvl
        -	id_class

	Таблица classes:
	      -	id
	      -	name
	      - abilities

    Таблица users_players:
        -	id
        -	id_user
        -	id_player

2. Выполните запросы в любой удобной для вас СУБД.
3. Заполните таблицы произвольными данными.
4. Напишите SQL запросы, которые выберут следующие данные из таблиц:

    -   Имена всех пользователей, возраст которых превышает 21 год.
    -	Среднее значение уровня всех персонажей
    -	Название класса персонажа, с самым высоким уровнем.

    user like "A%" OR user like "O%" or user like "%r%"

    -	Логин и хэш пароля пользователя, у которого ник начинается на букву ‘A’, ‘O’ или ‘U’ или содержит буквы ‘r’, ‘s’, ‘d’, ‘e’, ‘z’, ‘f’, ‘o’.
    -	Выведите полную информацию о пользователе по его id. Необходимо вывести
        (имя, фамилия., возраст, все персонажи (которые закреплены за этим пользователем), и их классы)
    -	Самый часто используемый класс персонажа.
